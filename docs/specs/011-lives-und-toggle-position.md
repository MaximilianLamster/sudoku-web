# 011 — Lives & Toggle Position

## Kontext
Die Herzen überlappen aktuell mit dem Sudoku-Inhalt, da sie in der fixen TopBar platziert sind. ThemeToggle soll immer oben rechts sein.

## Features

- [ ] Herzen erscheinen als eigene rechtsbündige Zeile direkt oberhalb des Sudoku-Grids (im Spielinhalt, nicht in der fixen TopBar)
- [ ] ThemeToggle bleibt fix oben rechts in der TopBar
- [ ] TopBar enthält nur noch BackButton (links) und ThemeToggle (rechts)

## Akzeptanzkriterien

- [ ] Keine Überlappung zwischen Herzen und Grid
- [ ] Herzen sind rechtsbündig über dem Grid ausgerichtet
- [ ] ThemeToggle ist auf Desktop und Mobile fix oben rechts sichtbar
- [ ] BackButton bleibt oben links
